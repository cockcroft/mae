MAE: Mathematica Accelerator Environment

Mathematica Accelerator Environment connects Mathematica to your favourite particle accelerator code. You can download the whole code in various ways, including clicking the link on the left.

What is MAE for?

MAE provides a way of controlling and connecting particle accelerator codes, in particular MAD, Elegant and GPT. It's a set of libraries (really, Packages and Notebooks) that do a lot of the 'heavy lifting' when you want to generate and optimise your accelerator design. It's mainly for high-energy accelerator design, but there is also an interface

Who is MAE for?

MAE is not for everyone. There is a learning curve, but it should pay off later on. Eventually there will be proper documentation and a paper on MAE, but for now you should know about these capabilities:

  • Generates input files for MAD, Elegant and GPT so they can be more easily compared;
  • Brings general simplex, genetic and other optimisers to accelerator design;
  • Allows very flexible visualisation of accelerator layout, optics, and particle tracking;
  • Makes using the underlying codes easier and more efficient .

Authors

MAE is being developed by physicists working mainly at the Cockcroft Institute. The lead developer is James Jones (STFC Daresbury), with support from Hywel Owen (Manchester University), John Jowett (CERN), Peter Williams (STFC Daresbury) and Julian McKenzie (STFC Daresbury).
